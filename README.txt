ОПИСАНИЕ 

бэк - Java, Spring Boot (MVC, Data, Security), Hibernate; БД - PostgreSQL; фронт - HTML, Bootstrap.

ИНСТРУКЦИИ

1. Для запуска приложения создайте базу данных factory, используя СУБД PostgreSQL
2. В application.properties измените username и password
3. Запустите FactoryCrmApplication
4. Зайдите в браузере по ссылке localhost:8080
5. Авторизуйтесь используя логин - admin и пароль - admin
6. С помощью вкладки Сотрудники(ADMIN) Вы можете создать новых пользователей,
обязательно наделите их ролями ENGINEER и MANAGER во вкладке Редактировать
7. Попробуйте доступный функционал для данных ролей
8. MANAGER может заполнять базу, редактировать ее, а также создавать новые Задачи и редактировать их
9. ENGINEER может пользоваться заполненной базой, а также заполнять и редактировать технологический процесс для Задачи
10. После заполнения технологического процесса MANAGER может списать материал, если он доступен в базе
и рассчитать стоимость обработки
11. Все пользователи могут оставлять сообщения в Чате
12. Для введения нового материала должны быть обязательно заполнены поля Диаметр х Длина, либо Высота х Шарина х Длина
13. При добавлении материала с характеристиками полностью совпадающими с уже имеющимся материалом -
его длина в базе увеличивается
14. Для введения нового инструмента Режущая часть обязательно должна быть меньше, чем вся длина инструмента
15. При добавлении инструмента с характеристиками полностью совпадающими с уже имеющимся инструментом -
его количество в базе увеличивается
16. Используя Профиль пользователи могут менять некоторые данные
